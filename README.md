# TARS Voice Server (Piper Edition) 🎤🚀

This is the server-side component of the TARS robotic assistant, responsible for fast and efficient Text-to-Speech (TTS) using Piper.

## 🌟 Features
- **Fast Response:** Uses Piper for near-instant speech synthesis.
- **Cross-Platform:** Works on Windows, macOS, and Linux.
- **Onnx Support:** Optimized for `TARS.onnx` models.
- **FastAPI:** Clean and easy-to-use API for the TARS Core.

## 🏗️ Architecture
The server listens for speech requests on the `/speak` endpoint and returns a `.wav` file generated by Piper.

## 🚀 Setup

### 1. Requirements
- Python 3.10+
- **Windows:** `piper.exe` should be in `piper_cli/piper/` (included).
- **macOS:** Install Piper via Homebrew:
  ```bash
  brew install piper-tts
  ```
- **TARS Model:** Ensure `TARS.onnx` is in the root directory.

### 2. Installation
```bash
pip install -r requirements.txt
```

### 3. Run Server
```bash
uvicorn w-tts-server:app --host 0.0.0.0 --port 8000
```

## 🛠️ API Reference

### Generate Speech
- **URL**: `/speak`
- **Method**: `POST`
- **Payload**:
  ```json
  {
    "text": "Hello, I am TARS."
  }
  ```
- **Response**: `audio/wav` binary file.
